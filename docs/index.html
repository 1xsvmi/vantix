VANTIX

Real-time cybersecurity monitoring for Windows and Linux. Process intelligence, network IDS, threat detection — in one sleek dark desktop app.

⬇ Download Free ★ GitHub

Everything You Need

Six fully integrated security modules — all real data, no mock values.

⚡

Overview Dashboard

Live CPU and RAM charts updating every second. Network upload/download speed. Active process and connection counters at a glance.

⚙

Process Intelligence

All running processes with PID, CPU%, memory, user, and risk level. Auto-refresh every 2 s. Color-coded SAFE / SUSPICIOUS / DANGEROUS. Double-click to kill.

🌐

Network IDS

Every active TCP/UDP connection with local/remote addresses, status, and process name. Flags suspicious ports like 4444, 1337, 31337 automatically.

💾

Storage Intelligence

All disk partitions with visual usage bars — green below 70%, orange below 90%, red above 90%. Real-time read/write speed monitor.

🎯

Threat Intelligence

FireHOL Level-1 blocklist fetched from GitHub and cached locally for 24 hours. Custom IP blocklist. Export threat reports as JSON or CSV.

🔗

Risk Correlation Engine

SQLite event database logging all security events with timestamps. 24-hour risk graph. Overall threat score calculated from all six modules. Auto-save every 5 minutes.
